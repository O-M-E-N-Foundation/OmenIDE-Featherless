## Security

The O-M-E-N Foundation takes the security of Omen IDE and related repositories seriously.

**Please do not report security vulnerabilities through public GitHub issues.**

### Reporting a vulnerability

Email **security@omen.foundation** with:

- A description of the issue and impact
- Steps to reproduce or a proof of concept if available
- Affected versions / commit if known

We will acknowledge receipt and work with you on coordinated disclosure.

Issues that must not enter the autonomous agent path should be labeled `security` on GitHub (maintainers only). Those issues are never auto-implemented or auto-merged.

For the original Microsoft VS Code sources that this fork is based on, Microsoft's guidance remains at [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
